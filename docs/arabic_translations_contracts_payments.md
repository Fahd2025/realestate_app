### ContractFormModal

The following keys are used in the contract form:

- `newContract` / `editContract` - Form title
- `selectOwner`, `selectProperty`, `selectTenant`, `selectBuyer` - Dropdown labels
- `monthlyLeaseAmount`, `purchasePrice` - Amount fields
- `descriptionEn`, `descriptionAr` - Description fields
- `startDate`, `endDate` - Date fields
- `paymentSchedule`, `paymentFrequency` - Payment section
- `previewPaymentSchedule` - Preview button

## Localization Generation

After adding the translations, run:

```bash
flutter gen-l10n
```

This generates the `AppLocalizations` class with all the new translation methods.

## How to Use in Code

```dart
import 'package:flutter_gen/gen_l10n/app_localizations.dart';

// In your widget
final l10n = AppLocalizations.of(context)!;

// Usage examples
Text(l10n.newContract)
Text(l10n.selectOwner)
Text(l10n.monthlyLeaseAmount)
Text(l10n.paymentSchedule)
Text(l10n.payNow)
```

## RTL Support

Arabic translations automatically support Right-to-Left (RTL) layout when the app locale is set to Arabic. The Flutter framework handles the text direction automatically based on the locale.

## Testing

To test the translations:

1. **Switch Language:**

   - Go to Settings
   - Select Language
   - Choose Arabic (العربية)

2. **Verify Components:**

   - Navigate to Lease Contracts
   - Open the contract form
   - Check all labels are in Arabic
   - Open payment modal
   - Verify payment-related text is translated

3. **Check RTL Layout:**
   - Ensure text aligns to the right
   - Verify icons and buttons are mirrored appropriately
